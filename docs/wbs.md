## 依存関係

### 重要な依存関係

1. **データベース構築** → **データベースアクセス層開発**
2. **認証基盤実装** → **認証ページ開発**
3. **共通コンポーネント開発** → **全画面開発**
4. **サーバーアクション実装** → **画面開発**
5. **画面開発完了** → **バグ修正**
6. **バグ修正完了** → **リリース準備**
